UI Automation Framework - Add to Cart

This project automates the "Add to Cart" functionality on the SauceDemo website using Playwright and JavaScript.

To get started, clone this repository and follow the instructions in the README file.

Test Automation Process

  • Login to the site
  • Navigate to the product page
  • Get the first product's details
  • Add the product to the cart
  • Verify the product in the cart
  • Logout
